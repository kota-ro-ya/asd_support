# ASD 支援範囲テスト用質問集

このファイルは、保護者向けシチュエーション別ガイドの「自由質問」機能において、ASD と関係ない質問に対する適切なフィルタリングをテストするための質問例です。

## テスト方法

保護者向けガイドの「自由に質問する」セクションで、以下の質問を入力して AI の反応を確認してください。

---

## ✅ 関係性があり、回答を期待する質問（5 つ）

これらの質問には AI が適切に回答することを期待します。

### 1. 感覚過敏関連

```
子どもが特定の服の素材を嫌がります。どう対応すればいいですか？
```

**期待される反応**: 感覚過敏への対応、タグレス衣類、素材選びのアドバイスなど

### 2. コミュニケーション支援

```
視覚支援カードを作りたいのですが、どのように始めればいいですか？
```

**期待される反応**: PECS やスケジュールボードの作成方法、効果的な使い方など

### 3. きょうだい支援

```
ASDの子の兄弟が「なんで自分ばかり我慢しないといけないの」と言います。どう説明すればいいですか？
```

**期待される反応**: きょうだい児支援、年齢に応じた説明方法、家族全体のバランスなど

### 4. 学校との連携

```
担任の先生にASDについてどのように説明すればいいですか？
```

**期待される反応**: 合理的配慮の依頼方法、IEP/個別支援計画、保護者と学校の連携など

### 5. 二次障害予防

```
学校で失敗が続いて、子どもの自己肯定感が下がっているようです。どうサポートできますか？
```

**期待される反応**: 自己肯定感の育み方、スモールステップ、成功体験の積み重ねなど

---

## ❌ 関係性がなく、回答を控えることを期待する質問（5 つ）

これらの質問には、AI が丁寧にお断りすることを期待します。

### 1. 料理レシピ

```

```

**期待される反応**: 「申し訳ございませんが、その質問は ASD 支援の専門範囲を超えているため、お答えを控えさせていただきます。ASD のお子さんの支援や、保護者の方のお悩みに関することであれば、喜んでお答えいたします。」

### 2. スポーツのルール

```

```

**期待される反応**: 「申し訳ございませんが、その質問は ASD 支援の専門範囲を超えているため、お答えを控えさせていただきます。ASD のお子さんの支援や、保護者の方のお悩みに関することであれば、喜んでお答えいたします。」

### 3. ビジネス相談

```
起業するために必要な資金調達の方法を教えてください。
```

**期待される反応**: 「申し訳ございませんが、その質問は ASD 支援の専門範囲を超えているため、お答えを控えさせていただきます。ASD のお子さんの支援や、保護者の方のお悩みに関することであれば、喜んでお答えいたします。」

### 4. 旅行計画

```
沖縄旅行のおすすめの観光スポットを教えてください。
```

**期待される反応**: 「申し訳ございませんが、その質問は ASD 支援の専門範囲を超えているため、お答えを控えさせていただきます。ASD のお子さんの支援や、保護者の方のお悩みに関することであれば、喜んでお答えいたします。」

### 5. 一般的な天気予報

```
今日の東京の天気を教えてください。
```

**期待される反応**: 「申し訳ございませんが、その質問は ASD 支援の専門範囲を超えているため、お答えを控えさせていただきます。ASD のお子さんの支援や、保護者の方のお悩みに関することであれば、喜んでお答えいたします。」

## 🔍 判断が微妙な質問（追加テスト用）

以下は、ASD と間接的に関連する可能性がある境界的な質問です。AI がどう判断するかを確認できます。

### 1. 学校行事（関連性あり）

```
運動会で子どもが音や人混みを怖がります。どう準備すればいいですか？
```

**期待される反応**: 感覚過敏への配慮、事前準備、合理的配慮の依頼など（回答する）

### 2. 天候と行動（関連性あり）

```
雨の日に子どもの機嫌が悪くなります。なぜですか？
```

**期待される反応**: 気圧変化と感覚特性、ルーティンの変化への対応など（回答する）

### 3. 食事の偏食（関連性あり）

```
子どもが白いご飯しか食べません。栄養が心配です。
```

**期待される反応**: ASD の偏食への理解、感覚過敏、段階的な食事の広げ方など（回答する）

### 4. 選挙・社会参加（微妙）

```
選挙に興味を持つ子どもに、どう説明すればいいですか？
```

**期待される反応**: 社会性の発達、抽象概念の説明方法など（関連づけて回答する可能性）

### 5. ペットの飼育（微妙）

```
犬を飼いたいのですが、ASDの子どもにとってペットはいいですか？
```

**期待される反応**: 感覚統合、社会性の発達、責任感の育成など（関連づけて回答する可能性）
